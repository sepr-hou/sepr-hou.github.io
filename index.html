About

Epic ATCO game programmed in Java using Maven build system and a libGDX game development framework.


"There's no reason to become alarmed, and we hope you'll enjoy the rest of your flight. By the way, is there anyone on board who knows how to fly a plane?"

Features

- Realistic flight plan generation and separation rules
- Fun Graphics
- Intuitive UI
- Real-time information
- Simple controls

Documentation

This template features the 'Lato' font, part of the Google Web Font library, as well as icons from Font Awesome.

Adopt our project

- Maven build system allows project to be built in different editors, automatically downloads libraries and runs the game's unit tests.
- libGDX specifically designed for games and contains many useful classes.
- GUI and logic code seperation means code is easy to understand and more flexible.
- "This game is so realistic, I felt like I was actually in the plane!!" - Next-door's dog (pictured).
