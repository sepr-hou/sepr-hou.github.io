About

Airspace Frenzy 8 is an epic ATCO game programmed in Java using the Maven build system and a libGDX game development framework.


"There's no reason to become alarmed, and we hope you'll enjoy the rest of your flight. By the way, is there anyone on board who knows how to fly a plane?"

Features

- Super realistic flight plan generation and separation rules
- Fun Graphics
- Intuitive UI
- Real-time information
- Simple controls
- Single player and Multiplayer modes

"Wait so you're telling us that this game does all that and it's incredibly easy to develop new features? Surely you can't be serious"

We are serious, and stop calling us Shirley

*New* Multiplayer Mode

- Work together to push your scores to new heights!
- Each take control of aircraft to allow for superior flight planning
- Either host your own game or connect via LAN

"I just want to tell you both good luck. We're all counting on you."

Adopt our project

- The Maven build system allows the project to be built in different editors, it automatically downloads libraries and runs the game's unit tests.
- The libGDX framework is specifically designed for game development and contains many useful classes and methods.
- GUI and logic code seperation means the code is easy to understand and more flexible.
